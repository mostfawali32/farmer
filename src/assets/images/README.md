# Images Folder

Place all your website images in this folder.

## How to use images:

1. **Add your images here** (e.g., `farm-field.jpg`, `logo-icon.svg`, etc.)

2. **Import in your component:**
   ```jsx
   import farmImage from './assets/images/farm-field.jpg'
   ```

3. **Use in JSX:**
   ```jsx
   <img src={farmImage} alt="Description" />
   ```

## Recommended image formats:
- **Photos**: `.jpg`, `.jpeg`, `.png` (use JPG for photos, PNG for transparency)
- **Icons/Logos**: `.svg`, `.png` (SVG is best for icons)
- **Optimize images** before adding them for better performance
